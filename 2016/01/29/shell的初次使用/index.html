shell的初次使用

Posted on 2016-01-29

shell的初次使用：

打开shell的方式：

1, 可以在桌面上右键单击，然后点open in terminal就可以打开一个shell了。
2, 可以点击图形界面上的Application -> system tools -> terminal 打开一个shell。
3，可以使用ctrl+alt+F2～F6 打开shell，不过请注意，这样打开的shell和以上打开的几种shell不同，以上几种都是在图形界面下打开shell，而此种则不是图形界面下的，想回到图形界面可使用快捷键ctrl+alt+F1/F7。
第一次接触shell：打开shell之后你会发现首行出现了类似与 [root@localhost ~]# 的提示
root 指的是打开此shell的用户名，在刚打开一个linux系统时，我们往往需要登录，在这个界面，我们选择登录的用户名并输入密码才能进入系统，需要说明的是在linux里，每个主机都有一个超级用户叫做root，现在我们可以先简单理解他的权限很高，可以对系统做很多修改，而普通用户不可以。其他的普通用户权限较低。
@ 就是一个分割符
localhost 指的是打开这个shell的主机名
～ 是此刻我们位于哪个目录下，先这么简单的理解，后面会有进一步的学习
# 是一个提示符，他它表示打开这个shell的用户是超级用户，若是普通用户开启shell则会是$符号。需要注意的是在使用shell的时候我们一定要清楚自己是哪个用户，处于哪个目录下。

初步使用shell：

我们知道，shell是我们与内核交互的工具，我们主要通过输入命令来让内核做相应的工作。大多数情况下我们输入的一行命令遵照“命令+参数+作用对象”的格式。
例如：wc -l /pub/file #wc 就是命令，-l是这个命令的一个参数，/pub/file 是一个文件是作用对象。

Shell中的一些快捷键：

tab键一下 ##补齐命令，命令参数，系统中的文件名
tab键两下 ##显示出所有可补齐的命令，命令参数或系统中的文件名
Ctrl+a ##将光标移动到行首
Ctrl+e ##将光标移动到行尾
Ctrl+u ##从光标所在位置删除到行首
Ctrl+k ##从光标所在位置删除到行尾
Ctrl+r 再输入关键字 ##搜索最近一条含有关键字的命令
Ctrl+左|右 ##光标向左|右移动一个单词的位置
Ctrl+c ##撤销当前输入的命令或正在执行的命令
Ctrl+d ##关闭当前shell
上下键 ##查看上一条命令或下一条命令如何查看shell中的历史命令
history ##查看历史命令
！数字 ##执行第n条命令
！关键字 ##执行最近一条以关键字的开头的命令Shell中常用的小命令
命令 –help ##查看命令的帮助文档##在帮助文档中：[内容]代表内容可加可不加。<内容>代表 内容必须加。内容…代表内容个数任意。
poweroff/init 0/shutdown -h now ##关机
reboot/init 6/shutdown -r now ##重启

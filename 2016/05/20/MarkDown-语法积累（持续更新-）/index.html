MarkDown 语法积累（持续更新--）

Posted on 2016-05-20

前言

MarkDown是用来写作的轻量级“标记语言”，它用简洁的语法代替排版，而不像我们在使用word时有大量的排版，字体设置。
它有便于我们专心于码字，编辑一篇文章时从内容到格式，插图，插入代码等等在键盘上就能搞定。
MarkDown 官方中文文档：http://wowubuntu.com/markdown/
本文主要用以记录在使用MarkDown语法格式写博客时积累的一些语法知识，将持续更新
